## Installatie ğŸ› ï¸
1. Installeer Node 18.14.0 LTS
2. Fork deze leertaak
3. Code klonen naar Github Desktop
4. Code open in VScode of andere editor
5. Terminal open
6. Npm install (typen in je terminal, daarna enter druken)
